# 🌊 Sunken Ruins

**신비롭고 위험한 바닷속 유적을 탐험하고, 다양한 심해 생물들의 위협을 피해 최종 목적지에 도달하세요!**
**Sunken Ruins**는 독특한 부스트 시스템과 슬로우 모션 메카닉을 특징으로 하는 2D 횡스크롤 플랫포머 게임입니다.

## 🎮 게임 개요

- **플랫폼**: PC (Windows)
- **장르**: 2D 플랫포머
- **Unity 버전**: 2022.3.27f1

## 🛠️ 게임 시스템

### 1. 실시간 스탯 및 자원 관리

- **체력**: 바닷속을 탐험하는 동안 체력이 지속적으로 감소합니다. 적에게 공격당하면 체력이 깎이고 잠시 무적 상태가 됩니다.
- **에너지**: 플레이어의 핵심 생존기인 **부스트**를 사용할 때 소모됩니다.

### 2. 조준 부스트 시스템

- **불릿 타임**: 마우스 클릭을 유지하며 조준하는 동안 화면이 플레이어 캐릭터에게로 줌인되고, 게임이 0.5배속으로 느려집니다. 이를 통해 긴박한 상황 속에서도 적의 공격을 정밀하게 피할 수 있습니다.
- **궤적 조준**: 조준 중 마우스 방향으로 날아갈 궤적이 그려져 정확한 방향을 잡도록 도와줍니다.
- **퍼펙트 부스트**: 게이지 바의 정확한 타이밍에 맞춰 마우스 클릭을 떼면 더 빠르고 멀리 이동하는 **퍼펙트 부스트**가 발동하며 이펙트 링이 방출됩니다.

### 3. 유용한 탐사 아이템

맵 곳곳에서 아래와 같은 유용한 아이템을 획득하여 생존 확률을 높일 수 있습니다.

- **체력 포션**: 급격하게 줄어든 체력을 대폭 회복합니다.
- **파워 배터리**: 에너지를 보충하여 더 많은 부스트 대시를 사용할 수 있게 해줍니다.
- **비눗방울 보호막**: 2초 동안 비눗방울 보호막이 켜지며 **무적 상태**가 됩니다. 이때 위험 구역을 안전하게 통과할 수 있습니다.

## 🦑 위협적인 심해 생물

바다 유적 곳곳에는 독특한 패턴과 공격 방식으로 플레이어의 앞길을 가로막는 다양한 적들이 도사리고 있습니다.

- **전기 가오리**
  - **특징**: 플레이어를 감지하면 빠른 속도로 쫓아옵니다. 일정 거리까지 근접하면 전기 에너지를 차징한 뒤 강력한 전기 공격을 가합니다.
  - **대처법**: 감전되면 **마비** 상태가 되어 일정 시간 동안 부스트 사용이 불가능해집니다. 전기 충전 범위가 보일 때 신속하게 부스트로 피해야 합니다.

- **최면 갑오징어**
  - **특징**: 플레이어에게 다가와 최면 광선을 발사합니다. 최면에 걸리면 이동 속도가 대폭 줄어들고 부스트를 사용할 수 없습니다.
  - **대처법**: 최면에 빠지면 QTE가 발동합니다. 화면의 지시에 따라 빠르게 키를 연타하면 탈출할 수 있습니다. 갑오징어가 몸통 박치기를 하기 전에 빠르게 연타하여 최면에서 벗어나야 합니다.

- **대왕 조개**
  - **특징**: 유적 바닥에 고정되어 있으며, 플레이어가 다가오면 강력하게 물을 흡수해 끌어당깁니다.
  - **대처법**: 조개 내부로 삼켜지면 껍데기가 닫히며 갇힙니다. 갇힌 상태에서 **제한 시간 내에 아무 키나 연타**하여 탈출하지 못하면 강력한 피해를 입습니다.

- **작은 게**
  - **특징**: 정해진 구역을 정찰하다 플레이어를 포착하면 제자리에 멈춰 서서 돌을 투척합니다.
  - **대처법**: 날아오는 돌의 궤적을 확인하고 타이밍에 맞춰 조준 부스트나 무빙으로 회피해야 합니다.

## 🗺️ 게임 흐름

1. **Title Screen**: 게임의 시작 화면입니다.
2. **Base Screen**: 기어 선택 및 맵 스크롤 화면입니다. 이곳에서 도전할 스테이지를 변경하고 진입합니다.
3. **Dive Scene**: 바닷속 유적으로 다이빙을 시작하는 컷신을 거쳐 스테이지에 진입합니다.
4. **Game Scene**: 수많은 적대적 바다 생물들 사이를 통과하며 앞으로 나아가야 합니다.
5. **최종 목적지**: 심해 바닥에 보관된 고대 유물 **간장**을 획득하면 화면이 페이드 아웃되며, 기지로 귀환하며 클리어합니다.
